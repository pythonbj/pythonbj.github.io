# 1.拼写错误

## 错误特征

Python报错：

```
NameError: name *** is not defined
```

## 错误描述

拼写错误，如```print```写成```puint```

## 改正方法

把错误的代码改正过来即可

# 2.缩进错误

## 错误特征


无缩进或缩进错误

## 改正方法

把错误的代码改正过来即可

## 3.无法输出中文（Python2）

## 错误特征

Python2报错：

```
SyntaxError: Non-ASCII character '\***' in file *** on line 1, but no encoding declared; see http://python.org/dev/peps/pep-0263/ for details
```

## 错误描述

Python2不支持中文

## 改正方法

改成Python3执行即可
